# PDF Math Translate

PDF Math Translate は、数式を含む PDF ファイルを翻訳するためのツールです。Ollama を使用して高品質な翻訳を提供し、数式の表示を維持したまま PDF を他の言語に変換することができます。

## 前提条件

- Node.js 18.x 以上
- Python 3.8 以上
- Ollama がインストールされていること
- 十分なディスク容量（フォントキャッシュ用）

## インストール

1. リポジトリのクローン：

```bash
git clone [repository-url]
cd pdfmathtranslate
```

2. Node.js の依存関係のインストール：

```bash
npm install
```
